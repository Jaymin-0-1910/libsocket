# libsocket

## Requirements

1. Android NDK r20 or up

## Building

1. Export NDK path to `ANDROID_NDK_HOME` variable
2. Run `build.sh` script in root of project folder

Binaries will be placed in `out/` folder and errors, if any, logged in `out/build.log`
file.
